News Bias Detector
This program uses the NewsAPI and VADER libraries to analyze the sentiment of articles from various news sources and determine if they are biased towards a particular political leaning.

Installation
Clone this repository to your local machine using git clone https://github.com/username/news-bias-detector.git.
Install the required libraries by running pip install -r requirements.txt in the project directory.
Obtain a NewsAPI API key by registering for a free account at https://newsapi.org/register.
Replace YOUR_API_KEY with your actual API key in the news_api_key variable in the main.py file.
Usage
Run the program by running python main.py in the project directory.
Enter the URL of the news article you want to analyze when prompted.
The program will output the average sentiment score of the articles from that news source and whether it is biased towards the left, right, or is unbiased.
